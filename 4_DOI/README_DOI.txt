4. DOI
-The package org.usc.edu.doi contains the source code that will consider the files in given dataset,identifies the MIME type and generates the short URL using YOURLS library.
-The package org.usc.edu.doi.mysql will help in retriving the data from the database including the short URLs.
-The database SURLs provided in the LISTING4 is required to be set up at the localhost server.
-Edit the /etc/hosts for to polar.usc.edu if required.
-The required database(SURLS) and YOURLS configured libraray is provided in the source code folder.
-The Sample_Output.doc is the JSON output of this step consisiting of DOI and the keyword for the given document.
