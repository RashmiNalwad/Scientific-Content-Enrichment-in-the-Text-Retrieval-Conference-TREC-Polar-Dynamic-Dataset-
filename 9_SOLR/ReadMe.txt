Apache Solr Set up:

1) Download Apache Solr 4.10.4.
2) Extract and Open command prompt in solr directory.
3) Run java -jar example/exampledocs/post.jar -c docs/
   where docs is a folder that contains the files to be indexed.(Files containing combined extracted metadata from steps 3,4,5,6,7)
=====================================================================================================================================
Schema.xml is the schema used for indexing data in Apache Solr.
